## 简介

本脚本旨在优化正方教务管理系统首页的证件照显示效果。启用后，可将正方教务管理系统首页显示的证件照，从入学后统一拍摄的照片，替换为入学前个人上传的证件照。同时，脚本会去除证件照的圆角和边距，恢复证件照的原始宽高比例，使照片展示更加真实、自然。

## 使用方法

1. 首先在浏览器中安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. 接着在 Tampermonkey 中安装 [ZFImageOptimize](https://greasyfork.org/scripts/498142) 脚本。
3. 安装完成后，访问教务系统首页即可自动体验优化效果，无需其他操作。

## 功能介绍

### 1. 证件照替换为入学前照片

首页原本显示的是入学后统一拍摄的证件照，通常较为生硬。脚本会自动切换为你入学前上传的证件照，因其由本人自主选择上传，通常效果更自然、美观。

### 2. 恢复证件照原始宽高比例

证件照默认被系统裁剪为正方形，容易导致人物形象变形、压缩。脚本会还原照片原始的宽高比例，让人物不再被拉伸或压扁，呈现最真实的照片状态。

### 3. 去除证件照圆角

系统原有的圆角设计会削弱证件照的正式感。脚本自动移除圆角处理，使证件照显示更加方正、规范。

### 4. 去除证件照边距

为避免照片周围出现多余的留白或间隔，脚本会去除证件照外部的所有边距，使照片与页面内容无缝衔接，整体观感更为紧凑协调。
